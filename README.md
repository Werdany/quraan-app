# quraan-app
# 📖 Quran Web App

## 🕌 Overview

A modern Islamic web application that provides an easy and beautiful way to read and listen to the Holy Quran, along with a collection of Hadith and daily Azkar.

The goal of this project is to create a simple, fast, and user-friendly experience for المسلمين في كل مكان.

---

## ✨ Features

* 📖 Read the Holy Quran (All Surahs)
* 🔊 Listen to Quran recitations
* 📜 Browse Hadith
* 🤲 Daily Azkar (Morning & Evening)
* 🔍 Search functionality
* 🌙 Dark Mode
* 📱 Fully Responsive Design

---

## 🛠️ Built With

* HTML5
* CSS3
* JavaScript (Vanilla JS)
* Public APIs (Quran Data)

2. Open the project folder

3. Run `index.html` in your browser
